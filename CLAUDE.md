## 3. How Giveaways Actually Work (Critical Industry Knowledge)

This context must inform all content, copy, and blog posts.

**The break-even formula:**
- Giveaway prize minimum: $50K+
- Need 3x prize value in revenue to break even
- 33% covers the prize, 33% covers ad spend, 33% covers COGS
- Once prize is paid off, profit margins explode
- This is why giveaways are high risk / high reward

**The reverse bell curve sales pattern:**
Giveaways follow a predictable 4-phase revenue curve:

1. **Launch Week** — Extremely high sales. First exposure to the new giveaway drives
   impulse purchases. Goal: maximize reach, scrape easy conversions, get the visual
   in front of as many prospects as possible.

2. **Middle Weeks** — Sales drop. Reduce ad budgets. Shift content from hard selling
   to nurturing, entertaining, and informational content.

3. **Mini Promos** — Additional promotions layered on top during middle weeks to spike
   revenue. Examples: "This weekend only, random Milwaukee tools in random orders" or
   "This week only, get entered for the truck AND the Can-Am Defender."

4. **Final Week** — Aggressive scaling. Scarcity messaging. Convert everyone nurtured
   during middle weeks + procrastinators. Sales spike in the final weekend.

**Entry multipliers (the giveaway currency system):**
Instead of running sales or discounts, we use entry multipliers. Instead of 20% off,
we offer 5,000 bonus entries. The multiplier has no margin cost but delivers real
perceived value to the customer. Clients effectively create their own currency — they
never need to run a sale again.

**Advertising philosophy:**
Giveaway advertising must be faster and more aggressive than traditional ecom.
Find winners fast, scale hard, kill losers immediately. Standard ecom agency pacing
will kill a giveaway campaign.

## 7. Design System

### Color Palette
Keep existing brand colors and logo. Apply a dark, intense, aggressive theme.

```css
:root {
  --color-bg:         #0A0A0A;      /* Near-black background */
  --color-bg-2:       #111111;      /* Card/surface background */
  --color-bg-3:       #1A1A1A;      /* Elevated surfaces */
  --color-accent:     #E8FF00;      /* Electric yellow-green — primary accent */
  --color-accent-2:   #FF4D00;      /* Burnt orange — secondary accent */
  --color-white:      #FFFFFF;
  --color-gray-1:     #F0F0F0;
  --color-gray-2:     #A0A0A0;
  --color-gray-3:     #444444;
  --color-border:     #222222;
}
```

**Design direction:** Dark, intense, performance-brand energy. Think luxury automotive
meets agency authority. Bold typography, high contrast, clean grid. NOT gimmicky,
NOT over-animated. Confident and premium.

### Typography
```css
/* Headlines: Bebas Neue or Barlow Condensed — bold, condensed, high-impact */
/* Body: DM Sans or Sora — clean, modern, highly readable */

--font-display: 'Bebas Neue', 'Barlow Condensed', sans-serif;
--font-body:    'DM Sans', 'Sora', sans-serif;
```

Load from Google Fonts:
```
https://fonts.googleapis.com/css2?family=Bebas+Neue&family=DM+Sans:wght@400;500;600&display=swap
```

### Typography Scale
```css
--text-hero:   clamp(72px, 10vw, 140px);   /* Hero headlines */
--text-h1:     clamp(48px, 6vw, 80px);
--text-h2:     clamp(32px, 4vw, 52px);
--text-h3:     clamp(22px, 2.5vw, 30px);
--text-body:   16px;
--text-small:  14px;
--text-label:  11px;  /* Uppercase labels, letter-spaced */
```

### Spacing
```css
--space-xs:   8px;
--space-sm:   16px;
--space-md:   32px;
--space-lg:   64px;
--space-xl:   120px;
--space-2xl:  180px;
```

### Component Patterns
- **Buttons (primary):** Solid `--color-accent` background, black text, no border-radius
  or very slight (2–4px). Bold, square-edged. On hover: slight scale or background shift.
- **Buttons (secondary):** Transparent background, 1px `--color-accent` border, accent text.
- **Section dividers:** Thin 1px `--color-border` lines or full-bleed accent bars.
- **Cards:** `--color-bg-2` background, `--color-border` border, no rounded corners OR
  very subtle 4px radius. Clean, utilitarian, premium.
- **Stats/numbers:** Display font, oversized, accent-colored. Let the numbers hit hard.
- **Labels:** 11px, uppercase, letter-spacing: 0.1em, `--color-gray-2`. Used above
  headlines to set context.
